Module Laze::Plugins::Less
In: lib/laze/plugins/less.rb

This plugin parses stylesheets written in Less to normal CSS files. The filename extension is converted to end in .css.

See lesscss.org for more information.

Note: when the less gem is unavailable this plugin will quietly fail, leaving the .less files untouched and generating a warning in the logs.
